# Projeto Movies

## 🎯 Objetivo do Projeto
Este projeto foi desenvolvido com o propósito exclusivo de **estudo**. Ele consiste em uma aplicação full-stack (frontend e backend) focada no consumo e gerenciamento de filmes e dados relacionados.

## 🛠️ Tecnologias Usadas

### Frontend (`/front`)
* **React** (v19) - Biblioteca para construção de interfaces de usuário.
* **Vite** (v8 Beta) - Ferramenta de build extremamente rápida para projetos web.
* **TypeScript** - Superconjunto de JavaScript que adiciona tipagem estática.

### Backend (`/back`)
* **Strapi** (v5) - Headless CMS poderoso e customizável criado com Node.js.
* **SQLite** - Banco de dados em arquivo, utilizado nativamente pelo Strapi para ambiente de desenvolvimento.
* **Axios** - Cliente HTTP para realizar requisições (como o consumo de APIs externas tipo a do TMDB).
* **TypeScript** - Tipagem utilizada também no ambiente Node para maior robustez de código.
